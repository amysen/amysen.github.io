Marketing Emotion Response Detector

This project aims to classify the emotion on a person's face when viewing advertisement media into one of seven categories, using deep convolutional neural networks. The model is trained on the FER-2013 dataset which consists of 35887 grayscale, 48x48 sized face images with seven emotions - angry, disgusted, fearful, happy, neutral, sad and surprised.
